# Cart-App-Using-ReactJS

> A shopping cart application built with React.js. 


## 🚀 Key Features

* **Dynamic Cart Management:** Instantly add, remove, or update the quantity of items in the cart.
* **Real-time Order Summary:** Automatically calculates subtotal, taxes, shipping, and the final total.
* **Persistent Storage:** Saves cart data in local storage so users don't lose their items on refresh.
* **Empty Cart States:** Beautiful UI handling for when the cart has no items.
* **Fully Responsive:** Flawless experience across mobile, tablet, and desktop screens.
* **Toast Notifications:** Smooth alerts for user actions (e.g., "Item added to cart!").
